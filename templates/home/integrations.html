Integratsiya Yo'riqnomasi

Ushbu qo'llanma orqali siz Coderboys ID xizmatini o'z loyihangizga OpenID Connect standarti asosida ulay olasiz.

OpenID Konfiguratsiyasi

Bizning servis standart .well-known/openid-configuration endpointini qo'llab-quvvatlaydi:

{ "issuer": "https://auth.coderboys.dev/o", "authorization_endpoint": "https://auth.coderboys.dev/o/authorize/", "token_endpoint": "https://auth.coderboys.dev/o/token/", "userinfo_endpoint": "https://auth.coderboys.dev/o/userinfo/", "scopes_supported": ["openid", "profile", "email"] }

1. Client kalitlarini olish

Integratsiyani boshlash uchun sizga Client ID va Client Secret kerak bo'ladi. Hozircha kalitlarni olish uchun administratorga murojaat qiling:

@axrorback bilan bog'lanish

2. Avtorizatsiya so'rovi

Foydalanuvchini Coderboys ID sahifasiga yo'naltiring:

GET https://auth.coderboys.dev/o/authorize/ ?response_type=code &client_id=YOUR_CLIENT_ID &redirect_uri=YOUR_CALLBACK_URL &scope=openid profile email

* redirect_uri bizning tizimda oldindan ro'yxatdan o'tgan bo'lishi shart.

3. Access Token olish

Callback URL orqali kelgan codeni tokenga almashtiring:

POST https://auth.coderboys.dev/o/token/ Payload: { "grant_type": "authorization_code", "code": "RECEIVED_CODE", "client_id": "YOUR_CLIENT_ID", "client_secret": "YOUR_CLIENT_SECRET", "redirect_uri": "YOUR_CALLBACK_URL" }

4. Foydalanuvchi ma'lumotlarini olish

Olingan access_token yordamida foydalanuvchi profilini yuklang:

GET https://auth.coderboys.dev/o/userinfo/ Headers: { "Authorization": "Bearer YOUR_ACCESS_TOKEN" }
Barcha so'rovlar HTTPS orqali amalga oshirilishi shart.
