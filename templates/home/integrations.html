Integratsiya Yo'riqnomasi

Coderboys ID platformasini loyihangizga OpenID Connect (OIDC) orqali ulab, markazlashgan autentifikatsiyani yo'lga qo'ying.

OIDC Discovery

Tizimimiz standart konfiguratsiya endpointini qo'llab-quvvatlaydi:

// .well-known/openid-configuration { "issuer": "https://auth.coderboys.dev/o", "authorization_endpoint": ".../o/authorize/", "token_endpoint": ".../o/token/", "userinfo_endpoint": ".../o/userinfo/", "scopes_supported": ["openid", "profile", "email"] }

1. Client kalitlarini olish

Hozircha tizim yopiq beta holatida. Client ID va Client Secret olish uchun administrator bilan bog'laning:

@coderboys_uz orqali so'rov yuborish

2. Avtorizatsiya so'rovi

Foydalanuvchini quyidagi manzilga yo'naltiring:

GET https://auth.coderboys.dev/o/authorize/
?response_type=code &client_id=YOUR_CLIENT_ID &redirect_uri=YOUR_CALLBACK_URL &scope=openid profile email

3. Access Token olish

Olingan codeni tokenga almashtirish uchun POST so'rovi yuboring:

POST https://auth.coderboys.dev/o/token/

Payload: { "grant_type": "authorization_code", "code": "RECEIVED_CODE", "client_id": "CLIENT_ID" }

4. Foydalanuvchi ma'lumotlari

Token orqali foydalanuvchi profilini JSON formatda oling:

GET https://auth.coderboys.dev/o/userinfo/
Headers: { "Authorization": "Bearer YOUR_TOKEN" }
HTTPS majburiy !
