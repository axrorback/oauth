Coderboys ID tizimining ishlash mantiqi va afzalliklari

Bu qanday servis?

Coderboys ID — bu markazlashgan autentifikatsiya va avtorizatsiya provayderi bo'lib, "Identity as a Service" (IDaaS) modelida ishlaydi. Siz o'z loyihangizda foydalanuvchilarni ro'yxatga olish, parollarni saqlash va login mantiqini yozib o'tirmaysiz. Barcha xavfsizlik va infratuzilma yukini biz o'z zimmamizga olamiz.

OAuth2 vs Oddiy API Auth

Nega shunchaki Login API emas, aynan OAuth2? Farqini jadvalda ko'ring:

Xususiyat Oddiy API (JWT) Coderboys OAuth2
Xavfsizlik Parol sizning bazangizda turadi Parol bizda, sizda faqat Token
SSO imkoniyati Faqat bitta saytda ishlaydi Barcha loyihalarga bitta login
Uchinchi tomon Xavfli (parol berish shart) Xavfsiz (faqat ruxsat olinadi)

Integratsiya Flow

1

Redirect

Ilovangiz foydalanuvchini bizning /authorize/ sahifamizga yuboradi.

2

Consent

Foydalanuvchi tizimga kiradi va sizning ilovangizga ma'lumotlarini olishga ruxsat beradi.

3

Callback

Biz foydalanuvchini maxsus bir martalik code bilan sizning saytingizga qaytaramiz.

4

Token Exchange

Sizning backend-ingiz ushbu kodni bizga yuborib, access_token oladi.

POST /o/token/ { "code": "...", "grant_type": "auth_code" }

Asosiy afzalliklar

Vaqt tejalishi: Auth tizimini 15 daqiqada tayyor qiling.
Xavfsizlik: Parollar sizning bazangizda saqlanmaydi.
SSO: Foydalanuvchi barcha loyihalaringizga bitta akkaunt bilan kiradi.
UX: "Sign in with Coderboys" tugmasi orqali tezkor kirish.
